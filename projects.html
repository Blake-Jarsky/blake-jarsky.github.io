Car Track Laps App

I'm building a web application that will help me find combinations of cars, tracks and race event lengths (laps) for a mobile car race game I play. https://race-game.herokuapp.com/

A player wiki website maintains a table of all game events, so I was able to build a relational database and query it via a node/express server hosted on heroku. I'm building the UI with react and styling with bootstrap.
